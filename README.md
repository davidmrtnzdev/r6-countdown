## üì¶ Installation

To run this project locally:

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/r6-countdown.git
    cd r6-countdown
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) to view the app.
